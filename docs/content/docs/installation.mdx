---

Currently, UI2 is only available as a JS/TS package. Install it with your favorite package manager!

```bash tab="NPM"
npm i ui2-sdk
```

```bash tab="PNPM"
pnpm add ui2-sdk
```

```bash tab="Bun"
bun add ui2-sdk
```

```bash tab="Yarn"
yarn add ui2-sdk
```
